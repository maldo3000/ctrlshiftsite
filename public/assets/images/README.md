# Images Directory

Place your image file here as `event-crowd.jpg` (or `.png`, `.webp`, etc.)

The image will be accessible via GitHub raw URL:
`https://raw.githubusercontent.com/maldo3000/ctrlshiftsite/main/public/assets/images/event-crowd.jpg`

After adding your image:
1. Add the file: `git add public/assets/images/event-crowd.jpg`
2. Commit: `git commit -m "Add event crowd image"`
3. Push: `git push origin main`

The image will then be available at the URL above and will display in the "Who We Are" section.
